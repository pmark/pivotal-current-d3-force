## Current Iteration

Edit .env and add Pivotal Tracker tokens.

```sh
PIVOTAL_PROJECT_ID=...
PIVOTAL_API_TOKEN=...
```

```sh
npm install
npm fetch
npm start
open http://localhost:8080
```
